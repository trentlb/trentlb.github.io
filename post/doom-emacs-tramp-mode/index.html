How do I edit files remotely in Doom Emacs with tramp mode?

Prerequisites

  1. Be running doom emacs.
  2. Have a remote server to connect to.

How do connect?

The default keybinding for emacs is C-x C-f, which opens a “Find file” buffer.

To reach the same buffer the doom emacs way, hit SPC f f.

Type this, replacing “username” and “server” with your own stuff. Don’t worry if the buffer is already prefilled with text. If you start with /, emacs will ignore the text before it.

/ssh:username@server:

This will drop you into Dired in the remote user’s home directory.

If you’d rather go straight to a file, use:

/ssh:username@server:/path/to/file

How do I easily reach the server again?

You can bookmark any file / directory that you can load. Opening the bookmark will automatically connect to the server.

In doom emacs:

SPC ENTER

Then enter your bookmark name and press ENTER again.

Any time you want to select this bookmark, type SPC ENTER.

How do I use a shell?

Enter a shell command like…

:eshell

:shell did not work for me, but it may work for you. I like the eshell, so I didn’t investigate further.
