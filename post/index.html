How do I edit files remotely in Doom Emacs with tramp mode?

published on May 12, 2021
Prerequisites Be running doom emacs. Have a remote server to connect to. How do connect? The default keybinding for emacs is C-x C-f, which opens a “Find file” buffer. To reach the same buffer the doom emacs way, hit SPC f f. Type this, replacing “username” and “server” with your own stuff. Don’t worry if the buffer is already prefilled with text.
